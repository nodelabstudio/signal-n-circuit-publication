# Author image bank

Store one approved portrait per author here.

Naming convention
- `<author-slug>.jpg`

Source of truth
- `/home/administrator/site/src/lib/authors.ts`

Rules
- Keep each file tied to the same author name permanently.
- Replace only with deliberate re-shoots that preserve the same identity.
- Recommended dimensions: 1200x1200 (square), high quality JPG.
