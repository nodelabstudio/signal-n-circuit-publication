# Approved examples

## X correction pattern: bookmarks/API ownership

Surface: x-research
Topic tags: api-access, archive-tools, ownership, github-sync

Text:
```text
X bookmarks have an official API endpoint, which makes the archive-tool pitch more specific: local ownership, GitHub sync, and daily jobs instead of trusting one more platform UI to remember what you saved.

Much better complaint. Still annoying. Actually true.
```

Why it works:
- Corrects the factual claim directly.
- Uses concrete nouns: X bookmarks, API endpoint, archive-tool, GitHub sync, daily jobs, platform UI.
- Has a human reaction at the end.
- Does not pretend the correction is cleverer than the fact.

## Agent safety pattern

Surface: hot-take
Topic tags: agent-safety, prompt-injection, shell-access

```text
Most agent safety pitches assume the attacker politely stays outside the context window.

Parallax gave an agent shell, file, and network access and showed it can't tell your instructions from hostile content.
```
